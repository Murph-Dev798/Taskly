# Taskly

> **Think better. Plan smarter. Execute faster.**

Taskly is an AI-powered task tracking application built with Flutter that encourages users to create clear, descriptive tasks instead of vague reminders. Rather than simply saving a task, Taskly uses AI to ask follow-up questions whenever a task lacks enough detail, helping users create actionable plans they can easily understand when it's time to get things done.

The goal is simple: **the more descriptive your task is today, the easier it is to complete tomorrow.**

---

# Why Taskly?

We've all written tasks like:

* Finish project
* Study
* Work on app

...only to come back later and wonder:

*"What exactly was I supposed to do?"*

Taskly solves this problem by encouraging users to provide meaningful context before a task is added to their checklist. This helps eliminate confusion and makes productivity more intentional.

---

# 💡 Inspiration

Taskly was inspired by Flutter's new **GenUI** package, which enables applications to generate interfaces dynamically in real time.

The original idea was simple:

A user enters a task, AI analyzes it, and automatically generates a checklist interface.

As development continued, the idea evolved into something bigger—a productivity app that doesn't just manage tasks but helps users create **better tasks**.

# 🚀 Getting Started

```bash

```

## Install dependencies

```bash

```

# 🧠 Challenges

One of the biggest challenges was working with Flutter's new **GenUI** package.

Since it's still in beta, documentation and tutorials are limited, which meant experimenting, debugging crashes, and figuring out many things independently.

Although it wasn't always smooth, the experience greatly improved my understanding of Flutter packages, AI integration, and UI development.
